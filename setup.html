🔧 Configuration de la synchro Google Sheets

Guide pas-à-pas pour connecter Villa Corsu Calendar à votre Google Sheet
1 Ouvrir le Google Sheet

Ouvrez votre Google Sheet dans le navigateur :

Ouvrir le Google Sheet →

Vérifiez que vous êtes bien connecté au compte Google propriétaire de ce Sheet.

2 Ouvrir Apps Script

Dans le Sheet, cliquez dans la barre de menus :

Extensions → Apps Script

Une nouvelle fenêtre s'ouvre avec un éditeur de code.

📋 L'éditeur affiche par défaut :
function myFunction() { }
3 Coller le script

Dans l'éditeur :

Sélectionnez tout (Ctrl+A ou Cmd+A) → supprimez → collez le code ci-dessous
// Villa Corsu — Web App Google Apps Script
const SHEET_NAME = "";

function doGet(e) {
  try {
    const ss    = SpreadsheetApp.getActiveSpreadsheet();
    const sheet = SHEET_NAME ? ss.getSheetByName(SHEET_NAME) : ss.getSheets()[0];
    if (!sheet) throw new Error("Feuille introuvable");
    const rows   = sheet.getDataRange().getValues();
    const header = rows[0].map(h => String(h).trim().toLowerCase());
    const iArr  = findCol(header, ["arrivée","arrivee","arr"]);
    const iDep  = findCol(header, ["départ","depart","dep"]);
    const iName = findCol(header, ["locataire","nom","name"]);
    const iNat  = findCol(header, ["nationalité","nationalite","nat"]);
    const iVis  = findCol(header, ["visiteurs","vis","personnes"]);
    const iPrix = findCol(header, ["prix","price","montant"]);
    const data = [];
    for (let i = 1; i < rows.length; i++) {
      const row   = rows[i];
      const arr   = formatDate(iArr  >= 0 ? row[iArr]  : "");
      const dep   = formatDate(iDep  >= 0 ? row[iDep]  : "");
      const name  = iName >= 0 ? String(row[iName] || "").trim() : "";
      if (!arr || !dep || !name) continue;
      const nat  = iNat  >= 0 ? String(row[iNat]  || "").trim() : "";
      const vis  = iVis  >= 0 ? parseInt(row[iVis],  10) || 0   : 0;
      const prix = iPrix >= 0 ? parseFloat(String(row[iPrix] || "0").replace(",",".")) || 0 : 0;
      data.push({ arr, dep, name, nat, vis, prix });
    }
    return jsonResponse({ ok: true, count: data.length, data });
  } catch (err) {
    return jsonResponse({ ok: false, error: err.message });
  }
}
function findCol(header, candidates) {
  for (const c of candidates) {
    const i = header.findIndex(h => h.includes(c));
    if (i >= 0) return i;
  }
  return -1;
}
function formatDate(v) {
  if (!v) return null;
  if (v instanceof Date) return pad(v.getDate())+"/"+pad(v.getMonth()+1)+"/"+v.getFullYear();
  if (typeof v==="string" && /^\d{1,2}\/\d{1,2}\/\d{4}$/.test(v.trim())) return v.trim();
  if (typeof v==="number") { const d=new Date(Math.round((v-25569)*86400000)); return pad(d.getUTCDate())+"/"+pad(d.getUTCMonth()+1)+"/"+d.getUTCFullYear(); }
  const d=new Date(String(v)); return isNaN(d)?null:pad(d.getDate())+"/"+pad(d.getMonth()+1)+"/"+d.getFullYear();
}
function pad(n) { return n<10?"0"+n:String(n); }
function jsonResponse(obj) {
  return ContentService.createTextOutput(JSON.stringify(obj)).setMimeType(ContentService.MimeType.JSON);
}

Puis cliquez sur 💾 Enregistrer (icône disquette ou Ctrl+S).

4 Déployer comme Application Web
Bouton bleu « Déployer » (en haut à droite) → « Nouveau déploiement »

Dans la fenêtre qui s'ouvre :

Cliquez sur l'engrenage ⚙️ à côté de "Sélectionner le type" → choisir « Application Web »

Remplissez les champs :

📝 Description : Villa Corsu API
👤 Exécuter en tant que : Moi (votre email)
🌐 Personnes ayant accès : Tout le monde ← IMPORTANT
Si vous choisissez autre chose que "Tout le monde", la synchro échouera.
Cliquez « Déployer »

Google va demander d'autoriser l'accès → cliquez « Autoriser », choisissez votre compte Google, puis « Autoriser » à nouveau.

5 Copier l'URL générée

Après le déploiement, Google affiche une URL d'application Web :

🔗 https://script.google.com/macros/s/AKfycb...longcode.../exec
Copiez cette URL complète — elle est nécessaire à l'étape suivante. Si vous fermez sans la copier, retrouvez-la dans Déployer → Gérer les déploiements.
6 Coller l'URL dans data.js

Ouvrez le fichier data.js de votre dépôt GitHub et remplacez :

const APPS_SCRIPT_URL = "COLLER_VOTRE_URL_ICI";

par :

const APPS_SCRIPT_URL = "https://script.google.com/macros/s/AKfycb.../exec";

Sur GitHub : ouvrez data.js → icône ✏️ → modifiez → Commit changes.
